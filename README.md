## 📃 About
This repository contains a series of laboratory works completed as part of the "AI Systems" course.
## 📂 Contents
**Lab 1:** Python Algorithmic Fundamentals and Control Structures
- 1_1_5.py & 1_2_5.py: Conditional Branching and Validation
- 1_3_5.py: Iterative and Recursive Nested Calculations

**Lab 2:** Lists, Tuples, Dictionaries, and Sets in Python
- 1_1_5.py: Generating a list of random integers within a given range
- 1_2_5.py: Filtering negative elements (with and without list comprehensions)
- 1_3_5.py: Removing a specified element from a list (two approaches)
- 1_4_5.py: Inserting an element into a list without using insert()
- 1_5_5.py: Managing student groups using dictionaries and a menu-driven program
- 1_6_5.py: Simulating a guessing game using sets and logical conditions
- 1_7_5.py: Finding common and different elements of two lists using sets
## 🛠 Technologies
